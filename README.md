# 📊 IS 6403:1981 Soil Bearing Capacity Web Estimator

A full-stack premium web application for computing the safe bearing capacity of shallow foundations in accordance with Indian Standard **IS 6403:1981** and **IS 1904:1986**.

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/nikunik177-ship-it/is6403-bearing-capacity)

---

## ✨ Features
* **Live Dynamic Computations**: Automatically switch between general, local, and mixed shear failure modes based on soil properties.
* **Premium Design System**: Glassmorphism dashboard with highly detailed visual feedback (interactive charts and equation breakdowns).
* **Interactive Water Table Interpretation**: Real-time correction calculations ($W'_q$ and $W'_\gamma$) based on water level.
* **PDF Geotechnical Report**: Generate and download professionally formatted PDF calculation reports on demand.

---

## 🛠️ Stack
* **Backend**: FastAPI (Python)
* **Frontend**: HTML5 / CSS3 (Vanilla) / Vanilla JS
* **PDF Engine**: ReportLab
* **Server**: Uvicorn
